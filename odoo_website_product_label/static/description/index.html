Website Product Labels & Stickers on Odoo Shop

If you want to add some Stickers/Tags/Label on your product which shows on website i.e Special offers, Sale , 50% off, Special price.

Now you can do this with help of this plugin, we have added few labels and sticker already on our apps which you can use.

Also you can add other label and stickers too (its configurable ) you can link that with product and it will be visible on Shop.

key features:

  • With the help of this module you can attach labels on different images of your product.
  • With the help of this module you can put Eye-catchy labels on your product Image.
